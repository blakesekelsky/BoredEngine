# BoredEngine
**A 2D game engine using SDL2.**

## Dependencies
* **clang**
  - Ubuntu: ```sudo apt install clang```
* **SDL2, SDL2_Image**
  - Ubuntu: ``` sudo apt install libsdl2-dev libsdl2-image-dev ```

## Make
* Build binary ```make build```
* Run binary ```make run```
* Do both ```make build_run```
